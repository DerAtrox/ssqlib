SSQLib

Originally over at google code, it is now ported across to GitHub so that it remains alive.

Simple example

To get the server information, you can use the following:

using SSQLib;

public void GetServerInfo()
{
    SSQL query = new SSQL();
    ServerInfo serverInformation = query.Server(ip, port);

    //You can now look at the properties of serverInformation to see what information you can retrieve.
}
